# 🚀 UN-Util — SCP:SL EXILED Plugin

Плагин для **SCP: Secret Laboratory** на базе EXILED, добавляющий новые игровые механики, рандом, фан и немного хаоса на сервер.

---

## ✨ Основной функционал

- 🎁 Рандомные предметы для Class-D (фонарик, лампа, монетка)
- 👋 Приветственное сообщение при заходе игрока
- 🪙 Улучшенная система монетки (рандомные эффекты, награды и наказания)
- ☠️ Команда `.suicide`
- 🔁 Команда `.res` (респавн с шансом 50/50)
- 🔥 Включение Friendly Fire в конце раунда
- 🚨 Кастомные уведомления C.A.S.S.I.E при дезинфекции

## ⚙️ Команды

| Команда | Описание |
|--------|----------|
| `.suicide` | Убить себя |
| `.res` | Респавн за D-Class или Scientist (50/50, с ограничениями) |
